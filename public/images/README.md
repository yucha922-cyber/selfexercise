# セルフケア画像の入れ方

各エクササイズの画像は、このフォルダ内の **エクササイズごとのフォルダ** に置くだけで、
サイトに自動で表示されます（JSONファイルの編集は不要です）。

## 基本の運用（1枚完結画像）

解説・手順・注意点まで入った「1枚完結」の画像を、各フォルダに **1枚だけ**
（例: `01.png`）置く運用を基本とします。1枚だけのときはページ内で
トリミングされずに全幅表示され、タップで拡大できます。
画像の作り方（レイアウト・文字サイズ・オーバーレイのルール）は
`docs/one-sheet-image-guide.md` を参照してください。

複数枚置いた場合は、従来どおりサムネイルのグリッド表示になります。

## ルール

1. **置く場所**: `public/images/<エクササイズのスラッグ>/` の中
2. **並び順**: ファイル名の昇順（`01_〜.jpg` → `02_〜.jpg` …）
3. **キャプション**: ファイル名の「先頭の数字と記号」を除いた部分が、画像の下の説明文になります
   - 例: `01_開始姿勢.jpg` → キャプション「開始姿勢」
   - 例: `02_あごを水平に引く.jpg` → キャプション「あごを水平に引く」
   - 例: `03.jpg` → キャプションなし
4. **対応形式**: jpg / jpeg / png / webp / gif / avif
5. **表示枚数**: 最大8枚まで表示されます

## 補足

- 画像を差し替えるときは、同じファイル名で上書きするだけでOKです。
- 並び順を変えたいときは、ファイル名の先頭の数字を変えてください。
- JSONファイル（`src/data/selfcare/*.json`）の `images` に手書きで指定がある場合は、
  そちらが優先されます（例: `katakori-stretch` は手書き指定のまま）。
- 将来、動画に切り替える場合は JSON の `youtubeId` に YouTube の動画IDを入れると
  ページ上部に動画が埋め込まれます。
