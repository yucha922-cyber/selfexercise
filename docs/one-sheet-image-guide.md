# 1枚完結エクササイズ画像 作成ガイド

エクササイズ1本につき解説入りの画像1枚で完結させるためのテンプレート仕様です。
「胸筋のストレッチ」の見本画像をベースに標準化しています。

作った画像は `public/images/<スラッグ>/01.png` のように1枚だけ置けば、
ページ内でトリミングされずに全幅で表示され、タップで拡大できます。

---

## 1. キャンバスの基本

| 項目 | 推奨値 |
| --- | --- |
| サイズ | 横 **1600 × 縦 1067 px**（3:2）を基準に。縦は内容に応じて±150px可 |
| 外周余白 | 上下左右 **40px** |
| 背景色 | 白 または ごく淡いクリーム（#FFFDF7 など） |
| 書き出し | **WebP または PNG**、幅1600px、目安 **400KB以下** |
| ファイル名 | `01.png`（数字のみ。画像内に説明があるためキャプション不要） |

## 2. ゾーニング（レイアウトの区画）

```
┌────────────────────────────────────────────┐
│ [タイトル帯]  [サブコピー（効果の一言）]        │ ← ヘッダー 約100px
├─────────────┬─────────────┬─────────────┤
│ ○ 開始姿勢    │ × ダメな例    │ ○ 終了姿勢     │ ← パネル見出し
│ ［写真 4:3］  │ ［写真 4:3］  │ ［写真 4:3］    │
│ ・箇条書き3点  │ ・箇条書き3点  │ ・箇条書き3点   │
│ ┌ポイント┐    │ ┌注意ポイント┐ │ ┌目安┐        │ ← 枠ボックス
├─────────────┴──────┬──────┴─────────────┤
│ 期待できる効果（チェック3点）│ 注意事項（・3点）        │ ← フッター帯
└────────────────────────────────────────────┘
```

- **3カラム構成**: 「開始姿勢 → ダメな例 → 終了姿勢」の順。左から時系列＋間にNGを挟むと視線が流れる
- 動きが単純な種目（肩甲骨回しなど）は「動き①→動き②→NG例」の置き換えも可。ただし **○=緑 / ×=赤 の色分けルールは固定**
- 各カラムの箇条書きは **3点まで・各行20文字以内**
- フッター帯: 左に「このストレッチで期待できる効果」、右に「注意事項」。どちらも3点まで

## 3. 写真上のオーバーレイ（描き込み）ルール

| 要素 | 使い方 |
| --- | --- |
| **点線サークル（赤）** | 意識する部位・注目ポイントを囲む。1枚に1個まで |
| **実線の曲線矢印（赤）** | 動く方向・力の方向を示す。始点=動く部位、太さ8px前後 |
| **引き出しラベル** | 白地(不透明度90%)＋赤文字。点線の引き出し線で部位につなぐ |
| **円形バッジ** | 「20〜30秒キープ」など時間・回数。写真の右下角に半分重ねる |

置き場所の原則:

1. **顔と、伸ばしている部位（主役）は隠さない**
2. ラベルは壁・背景など情報の少ない場所へ置く
3. オーバーレイは **1枚の写真に合計2個まで**（多いと注目点がぼける）
4. NG例のラベルは「肩がすくんでいる」など **状態を短く名詞で** 書く

## 4. 配色・文字サイズ

### 配色（意味を固定する）

| 色 | 用途 |
| --- | --- |
| 緑（例 #2E7D32） | 正しい姿勢の「○」、ポイント枠、効果 |
| 赤（例 #C62828） | NGの「×」、注意枠、オーバーレイの矢印・サークル |
| 濃紺〜黒（例 #1A2333） | タイトル・本文 |
| クリーム/白 | 背景・パネル地 |

### 文字サイズ（幅1600px時の最小値）

| 要素 | サイズ |
| --- | --- |
| タイトル | 64〜72px 太字 |
| サブコピー | 32〜36px |
| パネル見出し（開始姿勢など） | 38〜42px |
| 箇条書き・ボックス内本文 | **28px以上（これ未満にしない）** |
| 写真上のラベル | 26px以上＋白フチか白地 |

スマホでは画像が約360px幅（1/4.4）に縮むため、28px未満の文字は拡大しないと読めません。
タップで拡大できるとはいえ、**箇条書きは28pxを下限** にしてください。

## 5. 撮影時のポイント

- 背景は白壁など無地に近い場所で統一（オーバーレイやラベルを置く余白になる）
- 服装は背景と反対の濃色だと輪郭が出て矢印・サークルが映える
- 3カットとも **同じ立ち位置・同じズーム** で撮ると横並びにしたとき比較しやすい
- NG例は誇張気味に（肩をしっかりすくめる等）。小さな差はスマホで伝わらない

## 6. サイトへの反映

1. 完成画像を `public/images/<スラッグ>/01.png` に置く（1フォルダ1枚）
2. コミット＆プッシュすると、そのエクササイズページに全幅で表示される
3. 画像内と同じ内容でも、JSONの `steps` / `cautions` は残しておく
   （検索・読み上げ・画像が読み込めない環境のために必要です）
